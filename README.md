# Eval_ESL_FG-TPMS-plates
This repository provides the numerical data and MATLAB source code for the development of a novel iterative functional approach for higher-order elevation of shear deformation functions. The proposed iterative-order shear deformation theory (ISDT) is applied to functionally graded cubic-symmetric TPMS plates under a comprehensive evaluation of single-layer plate theories. 

This code repository contains the following features:
- Isogeometric analysis (IGA) modeling for FG-TPMS rectangular plates
- Eight equivalent single-layer theories, including CPT, FSDPT, TSDPT, FoSDPT, GPT, RPT, quasi-3D GPT, and quasi-3D RPT
- Static bending analysis results of FG-TPMS plates
- Plot packages for NURBS-based modeling of 2D surfaces, particularly for displacement, strain, and stress fields of plates
